# 💖 Feliz Dia das Mães

Site simples e animado feito com carinho pra minha mãe.

## 📸 Como trocar as fotos

1. Abra a pasta `fotos/`
2. Coloque suas fotos lá com os nomes exatos: `foto1.jpg`, `foto2.jpg`, `foto3.jpg`, `foto4.jpg`, `foto5.jpg`, `foto6.jpg`
3. Pode ser `.jpg` ou `.png` (mas mantenha o nome `foto1.jpg`, `foto2.jpg`, etc — ou edite o `index.html` pra apontar pra outros nomes)
4. As fotos que você não colocar vão aparecer como "📸 Foto X aqui" — isso é normal, é só o placeholder

> Dica: fotos no formato retrato (mais altas que largas) ficam melhores, tipo 4:5 (ex: 1080×1350px).

## ✏️ Como mudar os textos

Abra o arquivo `index.html` num bloco de notas (ou VS Code) e procure pelos textos. Eles estão direto no HTML, é só editar e salvar. As 4 mensagens estão na seção `<!-- Mensagens -->`.

## 🌐 Como ver no navegador (sem internet)

Dá dois cliques no `index.html` que abre direto no navegador padrão.

## 🚀 Como subir no GitHub Pages (deixar público gratuito)

Veja o passo-a-passo no chat com o assistente — basicamente:

1. Cria um repositório novo no GitHub (ex: `dia-das-maes`)
2. Sobe esses arquivos
3. Ativa o GitHub Pages em **Settings → Pages → Branch: main**
4. Aguarda 1-2 minutos e o site fica em: `https://SEU-USUARIO.github.io/dia-das-maes/`
